配置文件 app.json

pages

  • 记录页面存放路径

window

window - 全局设置窗口的外观

①navigationBar 导航条（window节点配置）

propety type default-vaue detail
navigationBarTitleText String title
navigationBarBackgroundColor HexColor #00000 bg-color
navigationBarTextStyle String white 导航标题颜色，仅支black+white

②background 背景区（默认不可见，下拉才显示）（window节点配置）

propety type default-vaue detail
backgroundColor HexColor #ffffff 窗口背景色
backgroundTextStyle String dark 下拉loading的样式，仅支持dark+light

③页面主体区域，用来显示wxml的布局（window节点不配置）

propety type default-vaue detail
enablePullDownRefresh Boolean false 是否群居开启下拉刷新
onReachBottomDistance Number 50 上拉，页面距离底部50px

tabBar

设置底部导航tabBar效果
最少2，最多5

  • backgorundcolor
  • text
  • color
  • iconPath
  • selectedIconPath
  • selectedColor

tabBar节点配置

propety type seting default-vaue detail
position String bottom 仅支持bottom top
borderStyle String black tabBar上边框的颜色，仅black+white
color HexColor 50 默认 item文本颜色
selectedColor HexColor 选择item文本颜色
backgroundColor HexColor tabBar的背景色
list Array true tab页白浅的列表

  • 是否启用戏班的组件样式
